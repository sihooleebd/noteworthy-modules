# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability within a module (e.g., malicious code execution possibilities via Typst features), please report it to the main Noteworthy repository maintainers immediately.

**Official Repository**: `https://github.com/sihooleebd/noteworthy-modules`

Please verify you are using the official modules. We are not responsible for forked or modified versions of these modules distributed via third parties.
